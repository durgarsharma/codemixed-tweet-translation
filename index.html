Code-Mixed Sentiment Analysis: Evaluating LLM Translation Against Multilingual Transformers on Twitter Data

Durga Sharma, Rahul Johari
University School of Automation and Robotics, GGSIPU, New Delhi
Poster@CODS-COMAD'24 Code

Abstract

Hindi and English (Hi–En) code mixing is widespread across Indian social media and remains a major challenge for sentiment analysis and cyberbullying detection. Current multilingual models struggle with this type of data, often giving inconsistent predictions and showing lower confidence. In this work, 27,314 tweets directed at six Indian athletes across three sports (2013–2023) to understand how code mixing affects automated sentiment classification. It was found that 12% of tweets contain Hi–En code mixing, with significant differences across athletes, and that 48.4% of all tweets are negative, with Wrestling athletes receiving the highest levels of negativity. Compared the state of the art multilingual models (XLM-RoBERTa, mBERT) with an LLM based translation pipeline and found that these models show only fair agreement on code mixed tweets, highlighting the difficulty of the task. Human evaluation on 498 tweets provides gold standard labels and shows that both multilingual models reach only moderate performance. Overall, the results show the limitations of current multilingual systems for processing code mixed text and demonstrate that translation based workflows offer a strong alternative for sentiment analysis in linguistically diverse social media settings.

Key Results

State of the art multilingual models show extreme divergence: XLM-RoBERTa classifies 42.9% as neutral while mBERT classifies only 3.8%, revealing fundamental disagreement on code-mixed sentiment.

Against human gold standard labels, both XLM-RoBERTa and mBERT achieve only moderate performance (F1 ≈ 0.59) with catastrophic failure on neutral sentiment classification.

LLM based translation achieves 76.4% sentiment preservation with systematic improvements: 11.5% neutral → negative corrections enhance cyberbullying detection while maintaining minimal distortion.

Dataset

We collected 27,314 unique tweets directed at six Indian athletes across Cricket, Boxing, and Wrestling over a decade (2013-2023). The dataset exhibits substantial linguistic diversity: 87.84% English-only, 12.01% Hindi-English code-mixed, and 0.11% Hindi-only tweets. Code-mixing intensity varies dramatically by athlete, ranging from 0.5% (Vijender Singh) to 48.1% (Sakshi Malik), reflecting differences in fan demographics and communication preferences. For code-mixed tweets, the mean Hindi character ratio of 0.58 indicates balanced bilingual expression rather than superficial language switching. Each athlete's collection period spans six months during their peak media visibility, capturing authentic public discourse during critical moments in their careers. The dataset includes tweet IDs, timestamps, athlete mentions, and language classifications, enabling reproducible analysis of sentiment patterns in multilingual social media contexts.

To validate automated predictions and establish gold-standard labels, we conducted rigorous human evaluation on 498 tweets selected via stratified random sampling (83 per athlete). Three independent annotators labeled each tweet as positive, neutral, or negative following standardized annotation guidelines emphasizing overall emotional tone and contextual interpretation. Inter-annotator agreement reached moderate levels (Cohen's κ = 0.586, 75.5% raw agreement), reflecting genuine ambiguity in social media sentiment complicated by sarcasm, cultural references, and implicit context. Gold-standard labels were derived through majority voting: 66.5% of tweets achieved full three-annotator consensus, 31.5% reached majority agreement, and 2.0% required manual adjudication. The resulting gold standard reveals significant cyberbullying presence (48.4% negative, 39.6% positive, 12.0% neutral), providing reliable benchmarks for model evaluation and establishing realistic performance expectations for automated systems processing code-mixed content.
